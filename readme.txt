== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/tier-price-table` directory, or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WooCommerce->Settings Name screen to configure the plugin

After installing the plugin, set up your own settings
